  Aquí les van algunas indicaciones básicas para tocar la canción:
  _________________________________________________________________
 |                                                                 |
 |      Para que el tema suene la raja te recomiendo ejecutar      |
 |  el acorde y los cambios de  acorde en el mismo momento en que  |
 |  aparecen,por eso logré que la ubicación de cada acorde sea la  |
 |  precisa, tal como puse los acordes sobre cada sílaba de verso  |
 |   y sobre cada punteo, así te será más fácil ubicar el acento   |
 | musical y además, obtendrás la armonía y melodía similares a la |
 |         vesión original de los David Bisbal, disfrútala.        |
 |_________________________________________________________________|
   ___________________________________
  |                                   |    CUERDAS de la GUITARRA:
  | Clave Universal   Clave Americana |    E    A    D   G   B   E
  |                                   |    |    |    |   |   |   |
  |       DO  ============== C        |    MI   LA   RE  SOL SI  MI
  |       RE  ============== D        |  Grave                  Agudo
  |       MI  ============== E        |
  |       FA  ============== F        |
  |       SOL ============== G        |
  |       LA  ============== A        |
  |       SI  ============== B        |
  |___________________________________|

 (Acorde)    (Cuerdas)
    |       E A D G B E    Los números corresponden al traste de la guitarra
    |       | | | | | |    donde se pulsa la cuerda señalada, la letra'X'
    |       | | | | | |    significa que no se pulsa la cuerda indicada.
    |       | | | | | |
    |       | | | | | |
    C    :  X 3 2 0 1 0  ( Do Mayor )
    Cm   :  X 3 5 5 4 3  ( Do menor )
    C#m  :  X 4 6 6 5 4  ( Do menor Sostenido   )
    C#m7 :  X 4 6 4 5 4  ( Do menor Sostenido 7 )
    D    :  X X 0 2 3 2  ( Re Mayor   )
    D7   :  X X 0 2 1 2  ( Re Mayor 7 )
    D#   :  X 6 8 8 8 6  ( Re Mayor Sostenido )
    D#   :  X X 5 3 4 3  ( Re Mayor Sostenido )
    E    :  0 2 2 1 0 0  ( Mi Mayor   )
    Em   :  0 2 2 0 0 0  ( Mi menor   )
    F    :  1 3 3 2 1 1  ( Fa Mayor   )
    F7   :  1 3 1 2 1 1  ( Fa Mayor 7 )
    F#   :  2 4 4 3 2 2  ( Fa Mayor Sostenido )
    G    :  3 2 0 0 0 3  ( Sol Mayor   )
    Gm   :  3 5 5 3 3 3  ( Sol menor   )
    G#m  :  4 6 6 4 4 4  ( Sol menor Sostenido )
    Am   :  X 0 2 2 1 0  ( La menor   )
    A#   :  X 1 3 3 3 1  ( La Mayor Sostenido )
    B    :  X 2 4 4 4 2  ( Si Mayor   )
    B7   :  X 2 1 2 0 2  ( Si Mayor 7 )
  ______________________________________________________
 |                                                      |
 |  Equivalencia de notas sostenidas con notas bemoles  |
 |______________________________________________________|
 |                                                      |
 |   C# = Db &lt;------&gt; ( Do  sostenido  =  Re  bemol )   |
 |   D# = Eb &lt;------&gt; ( Re  sostenido  =  Mi  bemol )   |
 |   F# = Gb &lt;------&gt; ( Fa  sostenido  =  Sol bemol )   |
 |   G# = Ab &lt;------&gt; ( Sol sostenido  =  La  bemol )   |
 |   A# = Bb &lt;------&gt; ( La  sostenido  =  Re  bemol )   |
